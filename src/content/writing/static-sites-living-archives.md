A static site does not need to feel frozen. The trick is to make editing as easy as opening a drawer.

## The pattern

- SvelteKit renders the site into static files.
- Markdown stores the content.
- Decap CMS gives you a browser editor.
- GitHub records every edit as a commit.

This keeps the public site fast while the private editing workflow stays flexible.
